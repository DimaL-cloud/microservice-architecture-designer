# Microservice Architecture Designer
AI-powered tool for designing microservice architectures 

## Setup
1. Clone the repository and navigate to the project directory.
2. Create copy of `.env.example` and rename it to `.env`. Fill in the required environment variables.
3. Run `docker-compose up` to start the application. Make sure you have Docker installed and running on your machine.
4. Access the application at `http://localhost:4200`.
